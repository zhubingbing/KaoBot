## 1. 产品概况

### 1.1 品牌定位

| 维度 | 描述 |
|------|------|
| **产品名称** | KaoBot |
| **一句话定位** | AI 驱动的考研复试全程智能辅导助手 |
| **品牌关键词** | 精准 · 高效 · 个性化 · 低焦虑 |
| **品牌主张** | "把复试准备的确定性还给每一位考生" |
| **品牌色** | 深蓝（专业/学术感）+ 橙金（活力/突破感） |
| **目标调性** | 既有导师的专业感，又有学长的亲切感；不说废话，直接给答案 |

**品牌故事**: 每年逾百万学生（2026 年约 174 万）经历初试后进入复试，面对 2-4 周极短窗口期，信息极度碎片化、备考策略高度不确定。KaoBot 用 AI Agent 将"导师信息、历年真题、简历定制、面试模拟"整合为一个对话入口，让每个人都能享受到原本只有"内部人脉"才能获得的备考资源。

> **数据来源说明（Data Sources）：**
>
> - **报名人数（2019–2026）**：教育部官方公告，经新华网、人民网转载。2026 年数据为教育部于 2025 年 11 月 24 日正式公布（[人民网报道](https://edu.people.com.cn/n1/2025/1124/c367001-40610589.html)）

> - **复试进入人数估算依据**：硕士招生计划 × 120%（教育部规定院校复试人数原则上不超过招生计划的 120%）。**注：34 所自划线院校（含北大、清华、复旦等）自主划定分数线，复试比例可达 130%–150%，竞争显著高于普通院校**；普通院校以国家统一划定 A/B 区分数线为准。
> - **数据截止日期**：2026 年 4 月。如有更新请参考 [教育部官网](https://www.moe.gov.cn) 及 [研招网](https://yz.chsi.com.cn)。

> **趋势解读**：报名人数自 2023 年峰值（474 万）已连续三年下降，2026 年降至 343 万（三年合计减少 **131 万人**）；但研究生招生计划逆势持续扩大，录取率从 2023 年谷底 27.5% 回升至 2026 年预计 42.3%，**复试竞争结构性调整：整体难度下降，但头部院校（34 所）竞争更加激烈**。

## 2. 产品体验五层模型

### 2.1 战略层

#### 用户需求地图

```
备考时间线：
─────────────────────────────────────────────────────────────────►
12月    1月      2月       3月初     3月中旬    3月末/4月
初试 → 等成绩 → 成绩出 → 国家线 → 复试通知 → 复试 → 录取
                  │         │         │
                  │         │         └── 需求爆发：真题、简历、面试模拟
                  │         └── 需求涌现：院校选择、调剂方案
                  └── 需求萌发：导师信息查询

复试阶段核心需求（KaoBot 主战场）：
┌──────────────────────────────────────────────────────────┐
│  1. 知道复试考什么（笔试方向、题型）                          │
│  2. 了解目标导师偏好（方向、风格、问题倾向）                   │
│  3. 做好简历（针对导师定制，不是通用简历）                    │
│  4. 准备自我介绍（中英文，控制时长）                          │
│  5. 模拟面试（降低临场焦虑）                                │
└──────────────────────────────────────────────────────────┘
```

#### 产品定位宣言

> KaoBot 是每一位考研生在复试阶段的"AI 学长"——它知道目标院校的复试规律，能帮你写简历、练口语，在最关键的 2-4 周里成为你最高效的备考伙伴。

#### 市场分析

**市场规模（TAM / SAM / SOM）**

| 市场层次 | 定义 | 规模估算 |
|---------|------|---------|
| **TAM** 总可触达市场 | 所有考研相关培训/工具市场 | ~500亿元/年 |
| **SAM** 可服务市场 | AI + 考研复试辅导细分市场 | ~15亿元/年 |
| **SOM** 实际可获取市场 | KaoBot Phase 1-2 目标市场 | ~3,000万元/年 |

**各层市场切割推导过程**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
TAM ≈ 500 亿元/年
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
计算公式：
  TAM = 中国考研培训市场整体规模（含初试+复试+工具）

数据来源与依据：
  · 艾瑞咨询《2024中国在线教育行业报告》：考研赛道市场规模约 480-520 亿元
  · 构成：网课/直播课（约55%）+ 线下班（约25%）+ 资料/工具（约10%）+ 一对一（约10%）

假设条件：
  A1. 市场规模采用 2024 年基准，未考虑考研人数下降带来的收缩（约-5%/年），
      保守处理以免高估机会
  A2. 包含初试+复试全阶段培训，未剔除非目标用户支出

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SAM ≈ 15 亿元/年
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
计算公式：
  SAM = TAM × 复试阶段支出占比 × AI工具付费渗透率
      = 500亿 × 15% × 20%
      = 500亿 × 3% = 15亿元

假设条件：
  B1. 复试阶段支出占全年考研支出比例 ≈ 15%
      依据：复试备考周期约 3-4 周（全年考研周期约 6-10 月），且复试课程均价低于初试
  B2. AI 工具在复试辅导中的付费渗透率 ≈ 20%（保守估算）
      依据：2024 年大学生 AI 工具月活渗透率 >60%，但付费转化约 20-30%；
            复试场景支付意愿高于日常，取 20% 作为保守基准
  B3. ⚠️ 敏感性分析：若 AI 渗透率升至 30%，SAM ≈ 22.5 亿；若降至 10%，SAM ≈ 7.5 亿

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SOM ≈ 3,000 万元/年（Phase 1-2，未来 24 个月）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
计算公式：
  SOM = 目标复试人群 × 目标渗透率 × 平均客单价
      = 174万人 × 1.15%渗透率 × ¥150 客单价
      ≈ 3,000 万元/年（24 个月目标）

假设条件：
  C1. 2026 年复试人群约 174 万（见 1.2 节数据）
  C2. Phase 1-2 市场渗透率目标 = 1.15%（即约 2 万付费用户）
      拆解：首年 500 付费用户（0.03%），次年 2 万（1.15%）—— 依赖口碑传播加速
  C3. 平均客单价 ¥150 = 复试包 ¥120（¥99 基础+¥199 旗舰加权均值）+ 月会员折算 ¥30
  C4. ⚠️ 核心风险假设：口碑传播裂变系数 K ≥ 1.5（每个付费用户带来 1.5 个新注册），
      若 K < 1，SOM 将收缩至约 800 万。需在 Phase 1 实测 NPS 后重新校准

对应理论上限：
  174万 × 30%（付费意愿群体）× ¥150 = 约 7.8 亿（5年终态上限，不作短期目标）
```

**市场趋势**

- 考研报名人数 2024–2026 年连续三年下降（三年减少 131 万），但招生计划持续扩大，录取率从 2023 年低点 27.5% 回升至 2026 年预计 42.3%，**复试竞争结构分化：头部 34 所自划线院校竞争加剧，普通院校录取率有所改善**

- 考研备考"个性化、精准化"需求显著增长，通用培训班性价比受质疑
- 复试"差异化"竞争加剧（国家线内的竞争者水平接近，复试差异决定胜负）

### 2.2 功能层

#### 基础功能

> 为所有用户免费提供，用于获客和建立基础信任。

| 功能 | 描述 | 限制（免费层） |
|------|------|-------------|
| 导师基础信息查询 | 搜索导师姓名、院校、研究方向 | 每日 5 次 |
| 复试问题速查 | 常见复试面试问题列表（非定制） | 无限制 |
| 考研日历提醒 | 国家线/复试通知/调剂时间节点提醒 | 无限制 |
| 帮助文档 / FAQ | 复试流程指南、注意事项 | 无限制 |

#### 核心功能

> 产品的价值核心，覆盖复试全流程。

| 功能模块 | 描述 | 关键特性 |
|---------|------|---------|
| **复试真题库** | F3: 按院校/专业/导师方向检索复试笔试题 | 历年题目 + AI 考点分析 + 备考计划生成 |
| **复试简历定制** | F4: 输入背景 → 生成针对导师的个性化简历 | 导师方向匹配 + 多版本生成 + 多轮优化 |
| **面试自我介绍** | F5: 生成中英文自我介绍脚本 | 1/2/3分钟版本 + 发音提示 + 对话式打磨 |
| **导师深度查询** | F1+: 导师论文、招生偏好、面试风格 | 结合知识库 + 实时搜索 |

#### 特色功能

> 形成差异化竞争壁垒的创新功能。

| 特色功能 | 描述 | 差异化价值 |
|---------|------|----------|
| **ReAct AI Agent 多工具串联** | 单次提问自动调用多个工具，一步获取复合答案 | 用户无需分多次提问，体验大幅优于 ChatGPT 直接提问 |
| **导师偏好画像** | 基于导师论文方向推断面试可能考察的方向 | 竞品无此功能，是 KaoBot 核心护城河 |
| **时间感知备考计划** | 输入距复试天数，自动生成最优复习安排 | 针对"时间极短"场景专项优化 |
| **复试包一键生成** | 填一次信息，同时输出简历 + 自我介绍 + 模拟题 | 减少重复录入，体验闭环 |
| **多语言面试模拟** | 中文/英文双语面试模拟，含学术术语发音注意点 | 针对有英文面试要求的顶校必备 |

**异常态设计通用原则**

| 原则 | 说明 |
|------|------|
| **有情绪** | 文案带有温度，避免机器感（"好像"、"抱歉"、"帮你"）|
| **给出路** | 每种异常态必须提供至少 2 个可执行的后续动作 |
| **不技术化** | 禁止展示 HTTP 状态码、堆栈信息、英文技术词汇 |
| **保护额度** | 因系统问题导致的失败不扣除用户使用次数 |
| **离线可用** | 网络异常时尽量展示已缓存内容（真题 PDF、历史对话）|

#### 整体设计风格

| 维度 | 设计决策 | 理由 |
|------|---------|------|
| **整体基调** | 清爽学术风（Academic Clean） | 对标 Notion / 语雀，用户群为高知学生 |

## 3. 行业分析

### 3.1 考研培训行业概况

**行业规模**：中国考研培训市场规模已超 **500 亿元**（2024年），年复合增速约 12%。

**行业特征**：
1. **重服务、轻技术**：传统玩家（新东方/考研帮/文都）以线下/线上课程为主，AI 工具渗透率低
2. **标准化 vs 个性化的矛盾**：大班课成本低但效果有限，一对一效果好但成本极高（500-2000元/小时）
3. **信息不对称严重**：复试信息（真题/导师偏好）高度分散，催生灰色信息中介市场

### 3.2 AI 教育工具趋势

| 趋势 | 现状 | KaoBot 机会 |
|------|------|------------|
| LLM 垂直化落地 | 通用 AI 无法精准回答"XX导师今年招几个" | 垂直知识库 + Agent 工具调用是关键 |
| 个性化学习需求升级 | 学生拒绝"千人一面"的复习材料 | 导师定制化是核心差异点 |
| 焦虑经济 | 高压备考催生大量"即时解决方案"需求 | 24小时可用的 AI 助手替代等学长回复 |
| 移动优先 | 90%+ 考研相关信息获取在手机上完成 | 移动端优先设计 |

### 3.3 政策环境

| 政策/趋势 | 影响 |
|----------|------|
| 研究生扩招政策持续 | 复试重要性提升，差异化竞争加剧 |
| AI 教育合规要求 | 需确保 AI 生成内容有来源标注，避免误导 |
| 个人信息保护法 | 用户上传文档需本地存储，不上传第三方 |
| 高校反作弊意识增强 | 简历/自我介绍生成功能需定位为"辅助"而非"代写" |

### 6.3 单位经济模型（Unit Economics）

> **为什么重要**：考研复试是高度季节性产品，LTV 窗口短、CAC 压力集中在 3 月爆发期。需在 Phase 1 跑通 CAC < LTV 的基本逻辑，才能判断规模化增长的可行性。

#### CAC（获客成本）各渠道估算

| 获客渠道 | 预估 CPM/CPC | 点击→注册转化率 | 注册→付费转化率 | **CAC（付费用户）** | 可扩展性 |
|---------|------------|--------------|--------------|-----------------|---------|
| 小红书 KOL 种草（腰部）| ¥500-1500/篇 | 约 3-5% | 约 8-12% | **¥250-600** | 中（依赖内容质量）|
| 知乎回答 + 问答引流 | 近乎 0 成本 | 约 2-3% | 约 5-8% | **¥0-50**（人力换算）| 高（长尾流量持续）|
| 考研帮/贴吧精准投放 | ¥0.5-2/点击 | 约 5-8% | 约 6-10% | **¥100-500** | 中 |
| 老用户口碑裂变 | 奖励成本 ¥20-50/单 | — | — | **¥20-50** | 极高（K 因子驱动）|
| 搜索广告（百度/微信）| ¥3-8/点击 | 约 8-12% | 约 5-8% | **¥500-1500** | 低（成本高）|
| **综合加权 CAC** | — | — | — | **约 ¥150-300** | — |

```
CAC 计算公式：
  CAC = 总获客营销支出 / 新增付费用户数

Phase 1 目标：
  CAC ≤ ¥200（以口碑 + 内容为主，避免付费广告为主要渠道）
  达成条件：口碑裂变 K > 1.0，即每个付费用户平均带来 1 个新注册
```

#### LTV（用户生命周期价值）

```
LTV 计算公式：
  LTV = 平均首次购买客单价 + 复购收入 + 转介绍价值

KaoBot 场景拆解（考虑考研一次性消费特性）：

  ┌── 当年复试用户
  │   首次购买：复试包 ¥99-199（均值 ¥130）
  │   追加：月度会员 ¥29.9 × 2月 = ¥59.8
  │   小计：LTV₁ ≈ ¥190
  │
  ├── 二战用户（次年复购）
  │   复购率估算：约 15-20%（考研二战率约 20%，其中使用 KaoBot 的复购率更高）
  │   次年贡献：¥190 × 18%复购率 = ¥34
  │
  ├── 转介绍价值（口碑传播）
  │   NPS 目标 >50 的前提下，每个付费用户转介绍约 0.8 个新付费用户
  │   转介绍价值（折现）= ¥190 × 0.8 × 50% 折扣因子 = ¥76
  │
  └── 综合 LTV = ¥190 + ¥34 + ¥76 ≈ ¥300

LTV/CAC 比值目标：LTV ¥300 / CAC ¥150 = 2.0×（健康基准为 ≥ 3×）
→ Phase 1 需将 CAC 压低至 ¥100 以下，或 LTV 提升至 ¥450+（通过调剂包交叉销售）
```

#### Payback Period（回收周期）

```
Payback Period = CAC / 月均用户贡献收入

月均用户贡献收入：
  = 复试包单次收入 ÷ 复试季时长（月）+ 月会员费
  = ¥130 ÷ 3月 + ¥29.9 ≈ ¥73/月

Payback Period = ¥150（CAC） / ¥73/月 ≈ 2.1 个月

⚠️ 季节性风险：
  考研复试季仅约 6-8 周（2月-4月），用户活跃窗口极短
  → CAC 需在 1 个复试季内回收（否则用户流失后无法续收）
  → 核心策略：将 CAC 压缩至 ¥100 以内，确保单季度回收

调剂应急包的 Unit Economics（更优）：
  · CAC：同渠道，但转化率更高（紧迫性驱动）≈ ¥80
  · LTV：¥65（调剂包单价）+ 复试包交叉销售 ¥50 ≈ ¥115
  · Payback Period = ¥80 / ¥115 < 1 个月 ✅（调剂包是利润率最优产品）
```

#### Phase 1 Unit Economics 达标条件

| 指标 | 目标值 | 当前估算 | 状态 | 关键杠杆 |
|------|--------|---------|------|---------|
| CAC | ≤ ¥100 | ¥150-300 | 🔴 待优化 | 提升口碑 K 因子至 1.0+ |
| LTV | ≥ ¥300 | ¥300 | 🟡 刚达线 | 交叉销售调剂包（+¥65）|
| LTV/CAC | ≥ 3× | 约 1-2× | 🔴 未达标 | CAC 降至 ¥100 方可达标 |
| Payback | ≤ 1 复试季 | 约 2 月 | 🟢 达标 | 维持现状即可 |

| 成本项 | 估算（月） | 优化方向 |
|-------|---------|---------|
| LLM API 费用 | ¥500-2000 | 缓存常见查询、模型量化降级 |
| Tavily 搜索 API | ¥200-800 | Phase 1 升级为本地知识库后大幅降低 |
| 服务器/云存储 | ¥300-800 | 初期轻量部署，按需扩容 |
| 开发人力 | 创始团队自承 | — |
| **合计** | **¥1,000-3,600/月** | LTV > CAC 拐点约在 500 付费用户 |

## 10. 未来展望与优化建议

### 10.1 短期优化建议（0-3个月）

| 优先级 | 优化点 | 预期收益 |
|--------|-------|---------|
| P0 | **PageIndex 替换 Tavily**：建立本地知识库，将真题、导师信息结构化入库 | 准确率从不可控 → ≥95%，知识护城河建立 |
| P0 | **简历 PDF 导出**：POC 只有文本建议，Phase 1 必须输出可投递的成品文件 | 完成真正的价值闭环，支撑付费 |
| P1 | **置信度标注**：所有 AI 回答标注信息来源和置信度（高/中/低） | 降低用户对 AI 幻觉的担忧，提升信任 |
| P1 | **用户档案持久化**：保存用户目标院校/导师/分数，避免重复填写 | 降低使用摩擦，提升留存 |
| P2 | **移动端适配**：当前 CLI 只能 PC 使用，移动端是主要使用场景 | 大幅扩大可用用户群 |

### 10.2 中期产品方向（3-12个月）

1. **知识库社区化**：引入用户贡献机制，让历届考生上传真题/经验，建立"考研 Wikipedia"
2. **导师画像深化**：通过 Semantic Scholar API 自动抓取导师近年论文，动态更新研究趋势
3. **模拟面试升级**：引入语音交互（STT+TTS），让面试模拟更接近真实场景，降低临场陌生感
4. **调剂信息实时化**：接入研招网 API / 爬虫，3月调剂季实现分钟级信息推送

### 10.3 长期愿景（1-3年）

```
愿景: 成为考研全周期的 AI 学业助手

备考期（9-12月） → 复试期（2-4月） → 入学后 → 求职期
   KaoBot            KaoBot            KaoBot    KaoBot
   初试规划           复试全程           学术辅导  求职助手
```

**扩展路径**：
- **横向扩展**：从考研 → 考博 → 出国留学 → 职场晋升（同样存在信息不对称 + 高焦虑的场景）
- **纵向扩展**：从复试辅导 → 入学后导师沟通助手 → 科研写作助手（陪伴用户整个研究生生涯）
- **平台化**：开放知识库给高校/培训机构，成为考研信息领域的"数据基础设施"

### 10.4 核心风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| 大厂入场（字节/百度做考研 AI） | 中 | 高 | 加速建立数据飞轮，做大厂不愿做的精细化运营 |
| AI 幻觉导致考生决策失误 | 中 | 高 | 所有信息强制来源标注 + 置信度标注，免责声明 |
| 真题版权纠纷 | 低 | 中 | 法律咨询，明确"用于学习研究"的合理使用边界 |
| LLM API 成本上涨 | 低 | 中 | 建立本地模型备用方案（Qwen/DeepSeek 本地部署） |
| 考研政策变化（如取消复试） | 极低 | 极高 | 产品设计保留向其他考试场景迁移的能力 |
